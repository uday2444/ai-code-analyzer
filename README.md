AI Code Analyzer is a tool built to analyze JavaScript and TypeScript projects using AI. Users can paste code, upload a file, or import a full project to get insights about the code. The tool helps identify possible issues and provides suggestions to improve code quality and structure.

The main focus of this project was to build the core functionality for analyzing code and understanding how AI can assist developers in reviewing codebases.

Note:
The UI was not a priority for this project. The goal was to focus on the functionality and logic behind code analysis rather than spending time on designing the interface.
